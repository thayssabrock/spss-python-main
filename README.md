# Spss Python

Este repositório reúne scripts em Python voltados ao tratamento de arquivos SPSS (.sav). Ele inclui funções para carregar dados, padronizar variáveis, converter tipos, mapear categorias para seus rótulos e identificar duplicidades. Também permite extrair metadados dos arquivos e selecionar .sav por meio de uma interface gráfica.

Entre os scripts disponíveis, há um voltado à visualização de informações gerais da base (checarbase.py) e outro focado na detecção de IDs duplicados e exportação dos resultados. O projeto está em constante expansão, com novas ferramentas planejadas para facilitar auditorias e verificações de dados. Dependências como pandas, pyreadstat e tk são necessárias para execução.
